What Are tables?

Tables were the only wat to orginize and display data, we didn't have div tags, TABLES SUCK! when we make a table to store data, we have to create the table row by row, cell by cell.

Tags used for Tables:

<table>:
Table Open and closing tags are used to tell HTML Browsers that we are building and placing a table in this location.
<tr>: Table Row
Creates The Indivisual Rows in a table.
<td>: td: Table Data
Create the indivisual cell within the rows we created, ans allows us to place text oe images within the tags.
<th>: Table Heading
The <th> that is used for table headers where it is automatically formated with bold text. replaces the <td> tag.
<thead>: Table Head
The Table Head is the section that you typically place table header information and tags.
<tbody>: Table Body
the main section of the table and hods the information
<tfoot>
The bottom of the table. Example is our total row.
<rowspan>
This Attribute is treated like merging cells merging/spanning over rows.
<colspan>
This Attribute is treated like merging cells merging/spanning over colomns.
